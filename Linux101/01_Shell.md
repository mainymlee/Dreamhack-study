# Shell

## Shell이란?

Shell은 사용자가 Linux 시스템을 사용할 수 있도록 도와주는
인터페이스이다.

사용자가 명령어를 입력하면 운영체제가 해당 명령을 처리하고, 처리 결과를
다시 사용자에게 출력한다.

``` text
User
   │
   ▼
 Shell
   │
   ▼
Operating System
```

------------------------------------------------------------------------

## Shell Prompt

Shell Prompt는 셸이 명령어를 입력받을 준비가 되었음을 나타낸다.

예시

``` bash
user@user-VirtualBox:~$
```

프롬프트가 표시되면 명령어를 입력할 수 있다.

------------------------------------------------------------------------

# id

현재 로그인한 사용자의 정보를 확인하는 명령어이다.

## 사용법

``` bash
id
```

## 출력 예시

``` text
uid=1000(user)
gid=1000(user)
groups=1000(user)
```

## 확인할 수 있는 정보

-   UID(User ID)
-   GID(Group ID)
-   사용자가 속한 그룹(Groups)

------------------------------------------------------------------------

# 핵심 정리

-   Shell은 사용자와 Linux 운영체제를 연결하는 인터페이스이다.
-   Shell Prompt가 나타나면 명령어를 입력할 수 있다.
-   `id` 명령어를 이용해 현재 로그인한 사용자의 UID, GID, 그룹 정보를
    확인할 수 있다.
