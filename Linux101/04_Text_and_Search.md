# 04. Text & Search

> Dreamhack Linux101 학습 정리

------------------------------------------------------------------------

# cat

파일의 내용을 출력하는 명령어이다.

## 사용법

``` bash
cat 파일명
```

### 예시

``` bash
cat hello.txt
```

------------------------------------------------------------------------

# file

파일의 종류를 확인하는 명령어이다.

## 사용법

``` bash
file 파일명
```

### 예시

``` bash
file /bin/ls
```

실행 파일, 텍스트 파일 등 파일의 유형을 확인할 수 있다.

------------------------------------------------------------------------

# echo

문자열을 출력하는 명령어이다.

## 사용법

``` bash
echo "Hello World"
```

리다이렉션과 함께 사용하면 파일에 내용을 저장할 수 있다.

``` bash
echo "Hello World" > hello.txt
```

------------------------------------------------------------------------

# grep

파일에서 특정 문자열을 검색하는 명령어이다.

## 사용법

``` bash
grep 문자열 파일명
```

### 예시

``` bash
grep root /etc/passwd
```

------------------------------------------------------------------------

# man

명령어의 매뉴얼을 확인하는 명령어이다.

## 사용법

``` bash
man 명령어
```

### 예시

``` bash
man cp
```

------------------------------------------------------------------------

# curl

서버와 데이터를 주고받는 명령어이다.

## 사용법

``` bash
curl URL
```

### 예시

``` bash
curl https://example.com
```

### 자주 사용하는 옵션

  옵션   설명
  ------ ---------------------
  `-o`   응답을 파일로 저장
  `-i`   HTTP 응답 헤더 포함
  `-X`   HTTP 메서드 지정
  `-d`   POST 데이터 전송

------------------------------------------------------------------------

# 핵심 정리

-   `cat`은 파일 내용을 출력한다.
-   `file`은 파일의 종류를 확인한다.
-   `echo`는 문자열을 출력하거나 파일에 내용을 저장할 수 있다.
-   `grep`은 원하는 문자열을 검색한다.
-   `man`은 명령어의 사용법을 확인한다.
-   `curl`은 서버와 데이터를 주고받을 때 사용한다.
