# Wildcard

Wildcard는 여러 파일이나 디렉토리를 한 번에 지정할 때 사용하는 특수
문자이다.

## 종류

  기호   설명
  ------ ---------------------
  `?`    임의의 한 문자
  `*`    0개 이상의 문자
  `[]`   문자 또는 범위 지정

## 예시

``` bash
cat h*
```

`h`로 시작하는 파일을 대상으로 명령을 수행한다.

``` bash
ls test[0-9]
```

`test0`부터 `test9`까지의 파일을 대상으로 한다.

# Redirection

Redirection은 명령어의 입력과 출력을 파일로 변경하는 기능이다.

## 출력 덮어쓰기

``` bash
echo "Hello" > hello.txt
```

파일이 없으면 생성하고, 있으면 기존 내용을 덮어쓴다.

## 출력 이어쓰기

``` bash
echo "Linux" >> hello.txt
```

기존 내용 뒤에 이어서 저장한다.

## 입력 받기

``` bash
grep root < passwd.txt
```

파일의 내용을 입력으로 사용한다.

# Pipe

Pipe(`|`)는 앞 명령어의 출력을 뒤 명령어의 입력으로 전달한다.

## 사용법

``` bash
명령어1 | 명령어2
```

## 예시

``` bash
ls -l | grep hello
```

현재 디렉토리 목록 중 `hello`가 포함된 항목만 출력한다.

------------------------------------------------------------------------

# 핵심 정리

-   Wildcard는 여러 파일을 한 번에 지정할 때 사용한다.
-   `?`, `*`, `[]`는 각각 다른 방식으로 파일을 선택한다.
-   `>`는 덮어쓰기, `>>`는 이어쓰기, `<`는 파일 입력에 사용한다.
-   Pipe(`|`)는 앞 명령어의 실행 결과를 다음 명령어의 입력으로 전달한다.
