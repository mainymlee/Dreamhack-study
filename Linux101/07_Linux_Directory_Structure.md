# Linux Directory Structure

Linux는 모든 파일과 디렉토리를 하나의 트리 구조로 관리한다.

가장 최상위 디렉토리는 **Root Directory(`/`)** 이다.

------------------------------------------------------------------------

## 주요 디렉토리

  디렉토리   설명
  ---------- -------------------------------
  `/`        최상위 루트 디렉토리
  `/bin`     기본 명령어 실행 파일
  `/boot`    부팅에 필요한 파일
  `/dev`     장치(Device) 파일
  `/etc`     시스템 설정 파일
  `/home`    일반 사용자 홈 디렉토리
  `/lib`     라이브러리 파일
  `/opt`     추가 프로그램 설치 위치
  `/proc`    프로세스 및 커널 정보
  `/root`    root 계정의 홈 디렉토리
  `/sbin`    시스템 관리 명령어
  `/tmp`     임시 파일 저장
  `/usr`     사용자 프로그램 및 라이브러리
  `/var`     로그 및 자주 변경되는 데이터

------------------------------------------------------------------------

# 디렉토리 설명

## /

Linux 파일 시스템의 시작점이다.

모든 디렉토리는 `/` 아래에 존재한다.

------------------------------------------------------------------------

## /bin

기본적으로 자주 사용하는 명령어가 저장되어 있다.

예시

``` text
ls
cp
mv
rm
cat
```

------------------------------------------------------------------------

## /boot

Linux가 부팅될 때 필요한 파일을 저장한다.

------------------------------------------------------------------------

## /dev

하드디스크, USB, 터미널 등 장치를 파일 형태로 관리한다.

------------------------------------------------------------------------

## /etc

시스템 설정 파일이 저장되는 디렉토리이다.

------------------------------------------------------------------------

## /home

일반 사용자의 홈 디렉토리가 생성되는 위치이다.

예시

``` text
/home/user
```

------------------------------------------------------------------------

## /lib

프로그램 실행에 필요한 라이브러리가 저장된다.

------------------------------------------------------------------------

## /opt

추가로 설치한 프로그램이 저장되는 위치이다.

------------------------------------------------------------------------

## /proc

현재 실행 중인 프로세스와 커널 정보를 확인할 수 있는 가상 파일
시스템이다.

------------------------------------------------------------------------

## /root

root 계정의 홈 디렉토리이다.

`/home`과는 별도로 관리된다.

------------------------------------------------------------------------

## /sbin

시스템 관리와 관련된 명령어가 저장된다.

------------------------------------------------------------------------

## /tmp

임시 파일을 저장하는 디렉토리이다.

------------------------------------------------------------------------

## /usr

사용자 프로그램과 라이브러리 등이 저장된다.

------------------------------------------------------------------------

## /var

로그 파일과 같이 자주 변경되는 데이터가 저장된다.

------------------------------------------------------------------------

# 핵심 정리

-   Linux 파일 시스템은 `/`를 기준으로 구성된다.
-   각 디렉토리는 서로 다른 역할을 가진다.
-   `/etc`에는 설정 파일이 저장된다.
-   `/home`에는 사용자 홈 디렉토리가 생성된다.
-   `/proc`은 프로세스 정보를 확인할 수 있는 가상 파일 시스템이다.
-   `/var`에는 로그와 같은 자주 변경되는 데이터가 저장된다.
