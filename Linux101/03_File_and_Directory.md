# 03. File & Directory

> Dreamhack Linux101 학습 정리

------------------------------------------------------------------------

# mkdir

새로운 디렉토리를 생성하는 명령어이다.

## 사용법

``` bash
mkdir 디렉토리명
```

### 예시

``` bash
mkdir new_dir
```

------------------------------------------------------------------------

# touch

빈 파일을 생성하거나 파일의 시간을 갱신하는 명령어이다.

## 사용법

``` bash
touch 파일명
```

### 예시

``` bash
touch hello.txt
```

------------------------------------------------------------------------

# mv

파일 또는 디렉토리를 이동하거나 이름을 변경하는 명령어이다.

## 사용법

### 이름 변경

``` bash
mv old.txt new.txt
```

### 다른 디렉토리로 이동

``` bash
mv hello.txt ../
```

------------------------------------------------------------------------

# cp

파일이나 디렉토리를 복사하는 명령어이다.

## 파일 복사

``` bash
cp hello.txt backup.txt
```

## 디렉토리 복사

``` bash
cp -r project backup
```

`-r` 옵션은 디렉토리를 재귀적으로 복사할 때 사용한다.

------------------------------------------------------------------------

# rm

파일이나 디렉토리를 삭제하는 명령어이다.

## 파일 삭제

``` bash
rm hello.txt
```

## 디렉토리 삭제

``` bash
rm -r project
```

`-r` 옵션은 디렉토리 내부의 파일까지 함께 삭제한다.

------------------------------------------------------------------------

# 핵심 정리

-   `mkdir` : 새로운 디렉토리를 생성한다.
-   `touch` : 빈 파일을 생성한다.
-   `mv` : 파일 이동과 이름 변경이 모두 가능하다.
-   `cp` : 파일과 디렉토리를 복사한다.
-   `rm` : 파일을 삭제하며, 디렉토리는 `-r` 옵션을 사용해 삭제한다.
