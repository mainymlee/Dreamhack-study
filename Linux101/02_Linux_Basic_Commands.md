# 02. Linux Basic Commands

> Dreamhack Linux101 학습 정리

------------------------------------------------------------------------

# apt update

설치 가능한 소프트웨어 패키지 목록을 최신 상태로 갱신하는 명령어이다.

## 사용법

``` bash
sudo apt update
```

------------------------------------------------------------------------

# apt upgrade

설치되어 있는 패키지를 최신 버전으로 업그레이드하는 명령어이다.

## 사용법

``` bash
sudo apt upgrade
```

------------------------------------------------------------------------

# pwd

현재 작업 중인 디렉토리의 경로를 출력하는 명령어이다.

## 사용법

``` bash
pwd
```

### 출력 예시

``` text
/home/user
```

------------------------------------------------------------------------

# ls

현재 디렉토리의 파일과 디렉토리 목록을 출력하는 명령어이다.

## 사용법

``` bash
ls
```

### 상세 정보 출력

``` bash
ls -l
```

`-l` 옵션을 사용하면 다음 정보를 함께 확인할 수 있다.

-   파일 권한
-   소유자
-   소유 그룹
-   파일 크기
-   수정 시간

특정 디렉토리의 내용을 확인할 수도 있다.

``` bash
ls -l /
```

------------------------------------------------------------------------

# cd

작업 중인 디렉토리를 변경하는 명령어이다.

## 사용법

루트 디렉토리로 이동

``` bash
cd /
```

홈 디렉토리로 이동

``` bash
cd ~
```

부모 디렉토리로 이동

``` bash
cd ..
```

------------------------------------------------------------------------

## 경로(Path)

### 절대 경로

루트 디렉토리(`/`)부터 시작하는 경로이다.

예시

``` text
/home/user
```

### 상대 경로

현재 위치를 기준으로 이동하는 경로이다.

예시

``` bash
cd ..
```

### 자주 사용하는 기호

  기호   설명
  ------ ---------------------------
  `~`    현재 사용자의 홈 디렉토리
  `..`   부모 디렉토리
  `-`    이전에 위치했던 디렉토리

------------------------------------------------------------------------

# 핵심 정리

-   `apt update`는 패키지 목록을 최신 상태로 갱신한다.
-   `apt upgrade`는 설치된 패키지를 최신 버전으로 업그레이드한다.
-   `pwd`는 현재 작업 중인 디렉토리의 경로를 출력한다.
-   `ls`는 파일과 디렉토리 목록을 출력하며 `-l` 옵션으로 상세 정보를
    확인할 수 있다.
-   `cd`는 디렉토리를 이동할 때 사용하는 명령어이다.
